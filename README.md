# NLP Project Practice Repository

This repository contains machine learning and NLP projects by **madhurkgp**.

## 📁 Projects

### Body Fat Percentage Estimator
A Flask-based web application that estimates body fat percentage using machine learning.

**Location:** `body_fat/`

**Features:**
- Clean, responsive web interface
- Real-time body fat percentage prediction
- Input validation and error handling
- Bootstrap-based UI

**Tech Stack:**
- Python 3.10+
- Flask 2.3.3
- Scikit-learn 1.3.0+
- Random Forest Regressor
- Bootstrap 4.5.0

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/madhurkgp/NLP_Project_Practice.git
   cd NLP_Project_Practice
   ```

2. **Navigate to the body fat estimator:**
   ```bash
   cd body_fat
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the application:**
   ```bash
   python app.py
   ```

5. **Access the app:** Open `http://localhost:5000` in your browser

## 📊 Project Details

### Body Fat Estimator
- **Algorithm:** Random Forest Regressor
- **Features:** Body density, abdomen, chest, weight, hip measurements
- **Dataset:** Body Fat Prediction Dataset (252 male subjects)
- **Accuracy:** R² score > 0.99 on test data
